Global Sales Performance Analysis

Tools Used:
    Excel • SQL • Power BI • DAX

Project Overview:
    This end-to-end data analytics project examines multi-country sales data to identify trends, measure business performance, and uncover actionable insights related to revenue, profitability, discounts, customer segments, and geographic distribution.

Key Analysis & Insights

   Sales & Profitability Performance
   Monthly & Daily Sales Trends
   Discount Impact on Profit
   Category-Level Performance
   Payment Method Behavior
   Location-Based Sales Insights

   DAX Measure Development
